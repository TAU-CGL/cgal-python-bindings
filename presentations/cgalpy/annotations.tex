\begin{frame}
  \frametitle{Type Annotation}
  \begin{itemize}
  \item Type annotations can be placed in external files called stubs
    \begin{itemize}
    \item They have the extension \lstinline{.pyi}
    \end{itemize}
  \item Our framework automatically generates the annotation stubs
  \item The framework includes a Python script that accepts as input static
    data files in the json format
  \item The input file describe the concepts, their refinement relations,
    the exposed C++ classes, and their modeling relations with concepts
  \item The script generates annotation stubs for bindings selected by
    the user
  \item The input files can be used to automatically generate the
    documentation for those models and concept...
  \end{itemize}
\end{frame}
