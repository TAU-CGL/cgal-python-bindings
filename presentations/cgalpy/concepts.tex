\begin{frame}
  \frametitle{Concepts-Binding Coupling}
  \begin{itemize}
  \item For each concept we have introduced a template function that generates
    bindings for all the type and function members required by the concept
  \item We call this function for every model of the concept
  \item This systematic approach guaranties that all the documented functions
    and types, but nothing else, are exposed
  \end{itemize}
\end{frame}
