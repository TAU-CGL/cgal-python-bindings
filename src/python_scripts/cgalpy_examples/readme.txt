The examples accept as an argument the name of the compiled bindings library to be used.

One precompiled library is provided for Windows 10 64 bit, Python 3.9 (CGALPY_kerEpecInt_arr2SegExPl_bso2_bv_ch2_pol2_ms2_pp_ss_tri2). If no argument is provided, this library will be used by default in all examples except those prefixed with "as3".
